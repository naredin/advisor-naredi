How To Use

Step-by-step instructions

Plug into OBD2 port with ignition ON.
Select AUTO SCAN for a full vehicle scan across all modules.
For oil reset: go to Service → Oil Service → follow on-screen instructions (usually takes 30 seconds).
For DPF regen: park safely, engine warm, select DPF Regeneration — takes 15–20 minutes.
For EPB: with wheels off the ground, select EPB Service → Retract for brake pad change.

FAQ

Common questions

Where is the OBD2 port in my car?
Almost always under the dashboard on the driver's side — look for a 16-pin trapezoid-shaped socket. In most Indian cars it is just to the left or right of the steering column, often without a cover.
Will reading fault codes damage my car's ECU?
No. An OBD scanner is read-only — it reads data the ECU is already storing. It cannot write to or modify your ECU. Clearing fault codes also only clears the stored memory, not any underlying issue.
If I clear the fault code, does the problem go away?
No — clearing the code only turns off the warning light. If the underlying issue is still present, the light will come back on within a few drive cycles. The correct use is: read the code, fix the problem, then clear the code to confirm it doesn't return.
What is the difference between a basic OBD scanner and a professional one?
A basic scanner (KW350, KW850) reads and clears engine fault codes only. A professional scanner (CS520, CS606) reads all systems — engine, ABS, airbag, transmission, TPMS — and can perform service resets like oil interval reset, EPB, and DPF.
